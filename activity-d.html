🧠 Why Does This Matter?

When a website gets hacked, attackers steal the database — including hashed passwords. Even if the hash can't be cracked, they can use credential stuffing: automatically trying the stolen username/password combo on Netflix, Gmail, banking sites, and more.

If you use the same password everywhere, one breach = all accounts compromised. The fix: use a different strong password for every site — and a password manager to remember them all.

Real-world victims include Adobe (153M accounts), LinkedIn (700M), Canva (137M), and RockYou (32M plain-text passwords).
